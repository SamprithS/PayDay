\# 💸 PayDay



A full-stack Progressive Web App to manage your monthly salary across Needs, Wants and Investments — so you always know exactly where your money is going.



\## Tech Stack

\- \*\*Frontend\*\*: Angular 19 (PWA) — deployed on Vercel

\- \*\*Backend\*\*: Java Spring Boot 3 — deployed on Render

\- \*\*Database\*\*: PostgreSQL (local + Render)



\## Structure


PayDay/

├── PayDay-frontend/ # Angular Progressive Web App (PWA)

└── PayDay-backend/ # Spring Boot REST API
